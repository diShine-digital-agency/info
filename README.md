## 🚀 Welcome to Our Digital Innovation Hub

Welcome to the official GitHub organization of **diShine Digital Agency**. Based in Milan, Italy, but operating globally, we are a creative tech agency dedicated to helping businesses and individuals navigate the complexities of the digital era. 

What started as a simple repository to share lists of tools has evolved into a comprehensive open-source hub. Today, this GitHub is becoming a place where we share the tools—modified and polished for public use—that have already improved our own workflows and those of our clients.

## 🛠️ What You'll Find Here: Our Open-Source Tools

This organization serves as our open-source contribution hub. We develop (and share modified version of) tools that we use internally and with our clients to automate workflows, audit performance, and build better tech stacks.

Here are our key open-source projects:

- **[Tools-and-Resources](https://github.com/diShine-digital-agency/Tools-and-Resources)**: an interactive toolkit featuring 426 digital tools with intelligent alternative matching. It allows you to build consulting tech stacks, find free/paid alternatives, and export triple-section reports.
  - *Tech Stack*: Standalone HTML/JS, Markdown, Astro (for dev server), JSON dataset. Zero dependencies.
- **[Prismo](https://github.com/diShine-digital-agency/prismo)**: the AI Consulting Toolkit. A portable, powerful, zero-installation USB drive toolkit bundling an AI-powered CLI with pre-built diagnostic prompts for website audits, SEO analysis, MarTech reviews, and security scans.
  - *Tech Stack*: Portable Node.js runtime, Claude Code (AI backend), PowerShell/Bash scripts.
- **[cookie-audit](https://github.com/diShine-digital-agency/cookie-audit)**: a GDPR compliance scanner that opens a site in a headless browser, captures every cookie, classifies them using a 150+ entry database, and outputs a graded compliance report.
  - *Tech Stack*: Node.js, Puppeteer (headless Chromium), CLI interface.
- **[tag-auditor](https://github.com/diShine-digital-agency/tag-auditor)**: an auditor for Google Tag Manager containers that finds unused tags, naming violations, missing consent, security risks, and performance issues.
  - *Tech Stack*: Node.js, JSON parsing, CLI interface.
- **[meta-inspector](https://github.com/diShine-digital-agency/meta-inspector)**: a tool to fetch any URL and inspect meta tags, Open Graph, Twitter Cards, and schema markup, simulating social previews across platforms.
  - *Tech Stack*: Node.js, HTML parsing, CLI interface.
- **[lighthouse-dashboard](https://github.com/diShine-digital-agency/lighthouse-dashboard)**: a self-hosted Lighthouse audit dashboard with scheduled monitoring and trend charts.
  - *Tech Stack*: Node.js, SQLite, REST API, Google Lighthouse.
- **[ai-prompt-library](https://github.com/diShine-digital-agency/ai-prompt-library)**: an expert prompt engineering library with CLI search, featuring 52 structured templates across various categories.
  - *Tech Stack*: Node.js (built-in modules only), YAML frontmatter, Markdown, HTML viewer.

## 🚀 Our Other Tools

Beyond our open-source contributions, we also build and maintain solutions designed to automate lead generation and process auditing at scale.

### [StreamLead.eu](https://streamlead.eu/)
**AI-Powered Lead Generation, Outreach & Lead Qualification Platform**

StreamLead turns fragmented web signals into a real-time feed of qualified prospects, then automates personalized outreach across 5 channels. It is an orchestration layer that combines the best data providers (Apollo, Hunter, Lusha), our proprietary AI agents, and custom algorithms into one unified, GDPR-native engine.
- **Key Features**: Smart Web Scraper (autonomous agents that extract data from any URL without CSS selectors), ICP Precision Matching, Intent Signal Radar, Smart Message Writer, and a Built-In Compliance Engine.
- **Use Case**: For sales teams and agencies who want to eliminate manual prospecting and ensure high deliverability with verified, enriched leads.

### [FlowAudit.it](https://flowaudit.it/)
**AI Process Audit & Workflow Optimization**

FlowAudit is an AI-powered platform that identifies exactly where automation can save your team time and money. By answering a few questions about your daily workflows, the platform generates a structured report detailing which tasks to automate, which tools to use, and the projected ROI.
- **Key Features**: 12-question diagnostic, AI-generated insights (powered by Gemini & Anthropic), concrete tool recommendations (e.g., Make.com, Zapier, n8n), ROI calculations, and premium document analysis via OCR.
- **Use Case**: For businesses looking to streamline operations, eliminate repetitive tasks (like CRM to ERP syncing), and get a clear, actionable roadmap for digital transformation.
